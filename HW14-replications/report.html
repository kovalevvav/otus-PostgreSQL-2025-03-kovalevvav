Репликация

    0. Создадим три ВМ с Postgres.

    1. На 1 ВМ создаем таблицы test для записи, test2 для запросов на чтение.

    ALTER SYSTEM SET wal_level = logical;
    CREATE TABLE test1(id INT NOT NULL, code TEXT NOT NULL);
    CREATE TABLE test2(id INT NOT NULL, code TEXT NOT NULL);

    2. Создаем публикацию таблицы test и подписываемся на публикацию таблицы test2 с ВМ №2.

    CREATE PUBLICATION  test1_pub FOR TABLE test1;

    CREATE SUBSCRIPTION test12_sub CONNECTION
    'host=178.154.223.102
     port=5432
     user=postgres
     password=12345
     dbname=postgres' PUBLICATION test2_pub WITH (copy_data = false, origin=none);

    3. На 2 ВМ создаем таблицы test2 для записи, test для запросов на чтение.

    ALTER SYSTEM SET wal_level = logical;
    CREATE TABLE test1(id INT NOT NULL, code TEXT NOT NULL);
    CREATE TABLE test2(id INT NOT NULL, code TEXT NOT NULL);

    4. Создаем публикацию таблицы test2 и подписываемся на публикацию таблицы test1 с ВМ №1.

    CREATE PUBLICATION  test2_pub FOR TABLE test2;

    CREATE SUBSCRIPTION test21_sub CONNECTION
    'host=158.160.56.252
     port=5432
     user=postgres
     password=12345
     dbname=postgres' PUBLICATION test1_pub WITH (copy_data = false, origin=none);

    5. 3 ВМ использовать как реплику для чтения и бэкапов (подписаться на таблицы из ВМ №1 и №2 ).

    ALTER SYSTEM SET wal_level = logical;
    CREATE TABLE test1(id INT NOT NULL, code TEXT NOT NULL);
    CREATE TABLE test2(id INT NOT NULL, code TEXT NOT NULL);

    CREATE SUBSCRIPTION test31_sub CONNECTION
    'host=158.160.56.252
     port=5432
     user=postgres
     password=12345
     dbname=postgres' PUBLICATION test1_pub WITH (copy_data = false, origin=none);
    CREATE SUBSCRIPTION test32_sub CONNECTION
    'host=178.154.223.102
     port=5432
     user=postgres
     password=12345
     dbname=postgres' PUBLICATION test2_pub WITH (copy_data = false, origin=none);

    5. Вставляем на 1 ВМ 10 записей в таблицу test1.

    INSERT INTO test1(id, code)
    SELECT generate_series as id, LPAD(generate_series::text, 2, '0') as code
    FROM generate_series(1,10);

    6. Вставляем на 2 ВМ 10 записей в таблицу test2.

    INSERT INTO test2(id, code)
    SELECT generate_series as id, LPAD(generate_series::text, 2, '0') || LPAD(generate_series::text, 2, '0') as code
    FROM generate_series(1,10);

    7. Убеждаемся, что после этих двух инсертов на всех трех ВМ в таблицах одинаковый набор записей.
